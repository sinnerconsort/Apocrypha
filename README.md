# 👁 Apocrypha

*It was always in the margins. You only started reading them.*

An eldritch attention engine for SillyTavern. Something has noticed your story. It accrues **attention** as you play — from your messages, from speaking of certain things, from your absences — and climbs through stages: **Unnoticed → Glimpsed → Regarded → Known → Named**. As it rises, it begins to whisper.

Whispers are **walled**: generated by an independent connection, rendered as faint asides beneath messages, completely invisible to the model and the story.

Until the wall breaks.

---

## v0.1 — the spine and the first organ

- **Attention engine** — hidden per-chat stat. Gains from every message you send, from trigger keywords, and from idle absence (it accrues while you're away; reopening the app after an hour is not free).
- **Whispers** — stage-tiered voice register, from broken 2–8 word fragments (Glimpsed) to fluent, intimate, name-knowing speech (Named). Anti-repetition memory: it remembers its last 8 whispers and is forbidden from echoing them.
- **Wall-break (experimental)** — at Named, the latest whisper leaks into real context at depth 2. Characters may half-hear it. Toggleable.
- **Debug eye** — FAB above the input bar. Shows attention, stage, gain log; lets you Feed +10, Force whisper, perform a Ritual (set stage floor), or Reset. Turn it off in settings once tuned — players of the finished thing should never see the number.

## Install

Drop the `Apocrypha` folder into:
```
SillyTavern/public/scripts/extensions/third-party/
```
Reload ST. Find **👁 Apocrypha** in the Extensions tab.

## Settings

| Setting | Meaning |
|---|---|
| **Hunger** | Attention gain multiplier. *Patient* (default) is a slow campaign-length burn; *Ravenous* hits Named in a one-shot. |
| **Default ritual** | Starting stage for **new** chats. Open a one-shot already Regarded. Per-chat override lives in the eye panel. |
| **Generation profile** | Connection profile for whisper generation. Point it at your utility profile — never burn your main API on this. |
| **Whisper cooldown** | Minimum AI messages between whispers. |
| **Trigger keywords** | Comma-separated. Speaking of these draws its attention (+5 each, capped per message). |
| **Wall-break** | Allow Named-stage whispers to leak into context. |

## Stage table (Patient hunger, no keywords)

| Stage | Threshold | Whisper chance |
|---|---|---|
| Unnoticed | 0 | — |
| Glimpsed | 20 | 25% |
| Regarded | 50 | 40% |
| Known | 95 | 55% |
| Named | 150 | 70% |

Tune thresholds/chances in `src/config.js` — everything is in one place.

## Known limits (v0.1)

- Whispers are keyed to message index; swiping or deleting messages can orphan/misplace them visually (state is unharmed — Reset clears render).
- Group chats untested.
- Reasoning models: budget defaults to 250 tokens; raise `maxWhisperTokens` in config if your utility model thinks before it speaks.

## Roadmap

- **The Witness** — walled asides that are *its perception of you*, Reliquary-style, unlocking at Regarded.
- **The Apocrypha organ** — fabricated lore: keyword-gated false World Info injected via `setExtensionPrompt`, so the model "knows" things nobody authored. Hybrid mode corrupts real Lexicon/lorebook entries when present.
- **Full wall-break** — staged leakage, not just Named; contaminated narration directives.
